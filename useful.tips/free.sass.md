# Free SaaS Alternatives

Remember, for every paid SaaS, there is a free open-source self-hosted alternative:

```text
Notion --> Appflowy
Zoom --> Jitsi
Jira --> Plane
Airtable --> NocoDB
Vercel --> Coolify
Heroku --> Dokku
Firebase --> Pocketbase/Appwrite/Convex/Supabase
Shopify --> Prestashop
GitHub --> GitLab
Slack --> Mattermost
Salesforce CRM --> ERPNext
Dropbox --> NextCloud
Mailchimp --> Mautic
Trello --> Wekan
Docusign --> Docuseal
Calendly --> Cal dot com
Datadog --> Prometheus
Google Analytics --> Matomo
Microsoft Office 365 --> LibreOffice
Asana --> OpenProject
```
